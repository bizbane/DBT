Install forge-1.17.1-37.0.71-installer.jar as a client.
Download Mods.zip extract the zip.
Click your windows button and type run and run the run program.
In the open window of the run program typ %appdata% click ok.
Click on .minecraft.
Find the mods folder and copy the files from the mod.zip into here.
If there is no mods folder create it and copy the files into this location.
Now open minecraft and join the DBTcraft server.
Info can be found in the DBT discord.
